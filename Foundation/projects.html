Here are a few projects I've worked on:


•ROTC email created in Salesforce for University of St. Thomas Students
    I made this using HTML and CSS in Salesforce while working in the IT department at my university. It was sent out at the beginning of the Fall semester to recruit students to the ROTC detachment at UST; It reached ~1300 students.




•Coffee Kiosk created in Java
    I made this in Java while in college. The purpose was to use multiple Java classes to implement an interactive coffee kiosk that allows you to customize a coffee, purchase a mug, display the total, and pay the amount displayed. It was probably one of my favorite projects in college, and was really neat to implement; It took 12 different files to create, and it involved using abstract classes, abstract interfaces, and classes that extend off others. The overall purpose was to showcase what we've learned in Object-Oriented programming, and create something a user could interact with.




•Epoch Time Converter in Java
   This was the first Java project I've ever made: An Epoch Time Converter. Made using one class, this program takes three inputs: A valid day, month, and year (If the date is invalid the program will output an error and close). It then converts that date into Epoch (Unix) time, or the number of seconds since January 1st, 1970; It can also go into the negatives if a date prior to 1970 is input. Being my first project, the code was nothing special, and the style was all over the place, but it has checks in place so that it computes dates correctly, and even outputs a random date from the year that was input.
